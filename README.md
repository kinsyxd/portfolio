# Портфолио-терминал

Одностраничное портфолио в стиле терминала. Интерактивный интерфейс с анимацией печати текста.

## Возможности

- интерфейс а-ля терминал linux
- текст появляется посимвольно с сохранением форматирования
- UI адаптирован под мобильные устройства

## Структура

```
portfolio_singlepage/
index.html      # разметка страницы
style.css       # стили и тема
script.js       # анимации и логика
background.webp # фоновое изображение
fonts/          # файлы шрифтов
icons/          # иконки для кнопок
```

## Технологии

- HTML5
- CSS3 (CSS-переменные, Flexbox)
- Vanilla JavaScript
- Google Fonts (Anonymous Pro, Ubuntu)
